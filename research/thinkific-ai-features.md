# Thinkific AI Features Research Notes

*Research Date: March 24, 2026*
*Sources: Thinkific Support Documentation, Thinkific Product Pages*

## Overview
Thinkific has developed a suite of AI-powered tools designed to streamline workflow, simplify tasks, and save time for course creators. These tools are built on Thinkific's own AI infrastructure.

## AI Tools Available

### 1. AI Course Outline Generator
**Purpose:** Speeds up course creation workflow
**How it works:**
- Takes inputs about target audience knowledge level, needs, and challenges
- Generates personalized, structured course outlines
- Outlines can be customized after generation
**Availability:** Available on all paid plans

### 2. AI Quiz Generator
**Purpose:** Creates dynamic quizzes from existing course content
**How it works:**
- Combines AI with existing course content
- Generates multiple-choice questions and answers based on chapter content
- Available within the quiz creation interface
**Availability:** Available on all paid plans

### 3. AI Landing Pages
**Purpose:** Quick creation of product sales pages
**How it works:**
- Automatically generates landing pages when products are created
- AI prompts guide page creation
- Web page editor allows customization of colors, text, imagery without coding
- Can be regenerated and customized at any time
**Available for:** Courses, Digital Downloads, Coaching & Webinars
**Key distinction:** Different from classic Site Builder pages - AI-generated pages use a newer web page editor

### 4. Email Automation with AI
**Purpose:** AI-driven marketing and sales automation
**How it works:**
- AI generates and tests email copy
- Generates subject lines and calls-to-action
- Personalized campaigns and automated sequences
- A/B testing capabilities for optimization
**Availability:** Growth + Premium plans and above

### 5. Closed Captions Autogenerator
**Purpose:** Accessibility and engagement improvement
**How it works:**
- Automatically generates SRT caption files for videos
- Works in the language of the speaker
- Processes directly within Thinkific video library
- No external tools needed
**Availability:** All paid plans

### 6. AI Dashboard Summarization (Thinkific Analytics)
**Purpose:** Faster data insights
**How it works:**
- Summarizes currently displayed dashboard data
- Provides bullet-point takeaways
- Helps consolidate findings and speed up data-to-insights workflow
**Availability:** Thinkific Plus and select plans

### 7. AI Course Builder (Beta - New)
**Purpose:** Modern, flexible course creation
**Key features:**
- Mix and match content types within single lessons (text, video, images, PDFs, quizzes)
- Live preview as you build (what you see = what learners see)
- Auto-save functionality
- Updated Course Player with modern design
- Block-based content editor

**New Course Builder vs Classic:**
| Feature | New Builder | Classic Builder |
|---------|-------------|-----------------|
| Lesson creation | Mixed content in one lesson | Separate lessons per type |
| Supported content | Text, image, video, PDF, quiz | Text, video, PDF, audio, multimedia, quiz, presentation, download, survey, exam, live |
| Saving | Auto-save | Manual save |
| Course Player | Updated modern design | Classic design |
| Preview | Exact learner view | Separate preview |
| Mobile app | Not yet supported | Supported |
| Custom code | Not yet supported | Supported |

**Current Limitations (Beta):**
- Not available on Thinkific mobile app
- No Shopify integration support yet
- Missing some settings: video % completion, free preview lessons, pre-requisites
- No course reviews or drip schedules
- No custom code snippets or third-party app extensions

## Key Competitive Differentiators
1. **Integrated approach:** AI tools built natively into the platform, not bolted-on
2. **No additional cost:** AI features included in plan pricing (no per-use fees)
3. **Content-aware:** Quiz generator actually reads your course content
4. **Accessibility focus:** Auto-captioning included at no extra charge
5. **Continuous improvement:** New course builder represents major AI-assisted evolution

## Limitations
- AI landing pages don't support all third-party apps that work with Site Builder
- New course builder in beta - some features still being added
- AI tools supplement, don't replace, strategic course design
