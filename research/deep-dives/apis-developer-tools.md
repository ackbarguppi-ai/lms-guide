# APIs & Developer Tools Research Notes

## Key Sources
- Thinkific Developer Platform: https://developers.thinkific.com/
- Thinkific Admin API: https://developers.thinkific.com/api/api-documentation
- Thinkific Webhooks API: https://developers.thinkific.com/api/webhooks-api/
- Thinkific REST API Intro: https://support.thinkific.dev/hc/en-us/articles/4422677315351
- Thinkific Webhooks Docs: https://support.thinkific.dev/hc/en-us/articles/4422658311703

## Key Findings
- Thinkific: REST API v2, Webhooks API, App Store, OAuth 2.0 for apps
- Docebo: REST + GraphQL, OAuth 2.0, extensive API coverage
- Absorb: REST API focused on enterprise workflows, batch operations
- TalentLMS: Straightforward REST API, well-documented
- Teachable/Kajabi/Podia: Limited to no API access
- SSO: Thinkific Expand+, Docebo, Absorb support SAML 2.0
