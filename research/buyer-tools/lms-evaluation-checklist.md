# LMS Evaluation Checklist

*A comprehensive checklist for evaluating Learning Management Systems. Use this during vendor demos, trial periods, and internal review meetings to ensure you're comparing platforms on what actually matters.*

---

## How to Use This Checklist

1. **Before you start:** Fill in your must-haves vs. nice-to-haves in the Priority column
2. **During demos:** Check off capabilities as vendors demonstrate them
3. **After trials:** Score each platform 1–5 on items that matter most
4. **Final decision:** Weight scores by priority to get your shortlist

**Priority Key:** 🔴 Must-have | 🟡 Important | 🟢 Nice-to-have

---

## 1. Core Learning Features

### Content Delivery
- [ ] Supports SCORM 1.2 and SCORM 2004 content packages
- [ ] Supports xAPI (Tin Can) for advanced tracking
- [ ] Native video hosting with adaptive bitrate streaming
- [ ] Supports multiple content types (video, PDF, HTML, audio, interactive)
- [ ] Drip/scheduled content release (e.g., release Module 2 on Day 7)
- [ ] Content locking and prerequisite enforcement
- [ ] Supports live/synchronous sessions (built-in or via integration)
- [ ] Offline access or downloadable content options
- [ ] Multi-language content support and localization

### Course Structure
- [ ] Flexible course builder (modules, lessons, sections)
- [ ] Learning paths / curriculum sequencing across multiple courses
- [ ] Cohort-based or time-bound course delivery
- [ ] Course templates for rapid creation
- [ ] Course cloning/duplication
- [ ] Bulk content import capabilities

### Assessments & Engagement
- [ ] Quiz engine with multiple question types (MCQ, fill-in, matching, essay)
- [ ] Question banks with randomization
- [ ] Graded assignments with rubrics
- [ ] Discussion forums or community features
- [ ] Certificates and badges (customizable templates)
- [ ] Gamification elements (points, leaderboards, streaks)
- [ ] Surveys and feedback collection
- [ ] Peer review capabilities

---

## 2. Content Authoring

- [ ] Built-in course authoring tool (no external software required)
- [ ] WYSIWYG editor for non-technical users
- [ ] Supports embedding third-party content (YouTube, Vimeo, Google Docs)
- [ ] Interactive content creation (H5P or equivalent)
- [ ] Screen recording or video capture built-in
- [ ] Co-authoring / multi-expert collaboration tools
- [ ] Version control for course content
- [ ] Content library for reusable assets (images, videos, documents)
- [ ] Accessibility authoring support (alt text prompts, contrast checks)
- [ ] Import from PowerPoint, Word, or PDF to course format

---

## 3. User Management & Roles

### User Administration
- [ ] Self-registration with approval workflows
- [ ] Bulk user import (CSV/Excel)
- [ ] Automated user provisioning via SCIM or API
- [ ] Custom user profile fields
- [ ] User groups and organizational units
- [ ] Sub-account or multi-tenant architecture
- [ ] User deactivation and data retention policies

### Roles & Permissions
- [ ] Granular role-based access control (not just Admin/Learner)
- [ ] Custom roles (e.g., Department Manager, External Partner)
- [ ] Instructor/facilitator role with limited admin access
- [ ] Manager role for team enrollment and progress oversight
- [ ] Delegated administration by department or business unit

### Enrollment
- [ ] Self-enrollment (open, coded, or approval-based)
- [ ] Auto-enrollment based on rules (role, department, hire date)
- [ ] Manager-assigned enrollment
- [ ] Enrollment deadlines and waitlists
- [ ] Batch enrollment for groups

---

## 4. Analytics & Reporting

### Standard Reports
- [ ] Course completion rates and progress tracking
- [ ] Individual learner transcripts
- [ ] Assessment scores and pass/fail rates
- [ ] Time spent per course/module
- [ ] Enrollment and engagement trends over time
- [ ] Certificate tracking and expiration alerts

### Advanced Analytics
- [ ] Custom report builder (drag-and-drop or query-based)
- [ ] Scheduled report delivery via email
- [ ] Export to CSV, Excel, or PDF
- [ ] Dashboard with real-time KPIs
- [ ] Cohort comparison and benchmarking
- [ ] Learning path progress tracking
- [ ] API access to raw reporting data
- [ ] Integration with BI tools (Tableau, Power BI, Looker)

### Compliance-Specific Reporting
- [ ] Audit trail for completions (timestamped, tamper-evident)
- [ ] Overdue training alerts and escalation
- [ ] Compliance status by team/department/location
- [ ] Recertification tracking and automated re-enrollment

---

## 5. Integrations

### Authentication & Identity
- [ ] SSO via SAML 2.0
- [ ] SSO via OAuth 2.0 / OpenID Connect
- [ ] LDAP / Active Directory sync
- [ ] Multi-factor authentication (MFA)

### HR & Business Systems
- [ ] HRIS integration (Workday, BambooHR, ADP, SAP SuccessFactors)
- [ ] CRM integration (Salesforce, HubSpot)
- [ ] Communication tools (Slack, Microsoft Teams)
- [ ] Calendar integration (Google Calendar, Outlook)

### Content & Conferencing
- [ ] Video conferencing (Zoom, Teams, Webex, GoTo)
- [ ] Content marketplace or off-the-shelf course library
- [ ] Third-party authoring tool compatibility (Articulate, Captivate)
- [ ] Document storage (Google Drive, SharePoint, Dropbox)

### Developer & Data
- [ ] REST API with comprehensive documentation
- [ ] Webhooks for event-driven automation
- [ ] Zapier / Make (Integromat) connector
- [ ] LTI 1.3 for tool interoperability
- [ ] Data export and portability tools

---

## 6. Mobile Experience

- [ ] Responsive web design (works on all screen sizes)
- [ ] Native mobile app (iOS and Android)
- [ ] Offline learning with sync on reconnect
- [ ] Push notifications for deadlines and new content
- [ ] Mobile-optimized assessments
- [ ] Video playback optimized for mobile (adaptive streaming)
- [ ] Mobile app branding (custom logo, colors)
- [ ] Feature parity between mobile and desktop (or close to it)

---

## 7. Branding & Customization

- [ ] Custom domain (e.g., learn.yourcompany.com)
- [ ] White-label branding (logo, colors, fonts)
- [ ] Custom login page
- [ ] Custom email templates and sender address
- [ ] Customizable learner dashboard
- [ ] Custom CSS/HTML access for advanced styling
- [ ] Multiple branded portals for different audiences (customers, partners, employees)
- [ ] Custom terminology (rename "courses" to "programs," etc.)

---

## 8. E-Commerce (If Selling Courses)

- [ ] Built-in payment processing (Stripe, PayPal)
- [ ] Subscription and membership pricing models
- [ ] One-time purchase and bundled pricing
- [ ] Coupon and discount code support
- [ ] Free trial or freemium options
- [ ] Tax calculation and invoicing
- [ ] Affiliate or referral program support
- [ ] Multi-currency support
- [ ] Checkout page customization
- [ ] Revenue reporting and payout management

---

## 9. Security & Privacy

### Data Protection
- [ ] Data encryption at rest (AES-256 or equivalent)
- [ ] Data encryption in transit (TLS 1.2+)
- [ ] Regular penetration testing with published results
- [ ] Data residency options (choose where data is stored)
- [ ] Data Processing Agreement (DPA) available
- [ ] Right to data portability and deletion

### Compliance & Certifications
- [ ] SOC 2 Type II certified
- [ ] GDPR compliant (for EU learners/data)
- [ ] FERPA compliant (for academic institutions)
- [ ] HIPAA compliant (for healthcare training)
- [ ] Section 508 / WCAG 2.1 AA accessibility compliance
- [ ] ISO 27001 certified
- [ ] CCPA compliant (for California residents)

### Infrastructure
- [ ] 99.9%+ uptime SLA
- [ ] Automatic backups with defined RPO/RTO
- [ ] Disaster recovery plan documented
- [ ] Status page with incident history
- [ ] Role-based access to admin functions
- [ ] IP allowlisting option
- [ ] Audit logs for admin actions

---

## 10. Scalability & Performance

- [ ] Handles your current user count without degradation
- [ ] Clear scaling path for 2x–10x user growth
- [ ] Concurrent user limits documented (or unlimited)
- [ ] CDN for global content delivery
- [ ] Page load times under 3 seconds
- [ ] Bulk operations perform well (10,000+ user imports)
- [ ] Multi-region deployment options
- [ ] Storage limits clearly defined (or unlimited)
- [ ] API rate limits documented and sufficient

---

## 11. Support & Success

### Vendor Support
- [ ] 24/7 support available (or hours that match your time zones)
- [ ] Multiple support channels (email, chat, phone)
- [ ] Defined SLAs for response and resolution times
- [ ] Dedicated account manager or customer success manager
- [ ] Priority support tier available
- [ ] Support team has deep product expertise (not just tier-1 scripts)

### Self-Service Resources
- [ ] Comprehensive knowledge base / help center
- [ ] Video tutorials and walkthroughs
- [ ] Community forum or user group
- [ ] Regular webinars and training sessions
- [ ] Developer documentation and API reference
- [ ] Template library for common use cases

### Onboarding & Implementation
- [ ] Dedicated implementation specialist
- [ ] Structured onboarding program with milestones
- [ ] Data migration assistance
- [ ] Train-the-trainer programs
- [ ] Implementation timeline fits your requirements
- [ ] Professional services available for customization
- [ ] Sandbox/staging environment for testing

---

## 12. Pricing & Contract

### Cost Structure
- [ ] Pricing model is clear and predictable (per user, per active user, flat fee)
- [ ] No surprise costs for storage, bandwidth, or API calls
- [ ] Free trial or pilot available
- [ ] Flexible contract terms (monthly, annual, multi-year)
- [ ] Volume discounts for large deployments
- [ ] Nonprofit or education pricing available

### Total Cost of Ownership
- [ ] Implementation/setup fees clearly stated
- [ ] Training costs for admins included or quoted
- [ ] Cost of premium support tiers documented
- [ ] Content migration costs estimated
- [ ] Integration development costs estimated
- [ ] Year-over-year price increase caps or guarantees

### Exit Terms
- [ ] Data export included at no extra cost
- [ ] Reasonable cancellation terms
- [ ] Data deletion timeline after contract ends
- [ ] No lock-in penalties
- [ ] Transition support available

---

## 13. Vendor Viability

- [ ] Company has been operating for 5+ years (or has strong backing)
- [ ] Regular product updates and public roadmap
- [ ] Healthy customer base in your industry/size segment
- [ ] Positive reviews on G2, Capterra, or TrustRadius
- [ ] Reference customers available to speak with
- [ ] Financial stability (funding, revenue, profitability indicators)
- [ ] Active user community and ecosystem
- [ ] Clear product vision aligned with your long-term needs

## Decision Checklist

Before signing:

- [ ] At least 2 stakeholders have seen a live demo
- [ ] You've completed a hands-on trial with real content
- [ ] You've spoken with at least 2 reference customers
- [ ] IT/Security has reviewed and approved
- [ ] Legal has reviewed contract and DPA
- [ ] Total cost of ownership is calculated for Year 1 and Year 3
- [ ] Implementation timeline is agreed and realistic
- [ ] Exit/migration terms are acceptable
- [ ] The platform genuinely solves your top 3 problems
