# Thinkific Mobile Learning Research Notes

## Thinkific Mobile App
- Available on iOS and Android
- Complement to web experience (not full replacement)
- Students can: access lessons, communities, receive push notifications
- Students cannot on mobile: purchase courses*, live events, update account, certificates
- *In-App Purchases available for Branded Mobile apps only
- No offline access currently (working on it — downloads of videos/PDFs/audio available)
- Progress syncs between mobile and web
- Webhooks fire from mobile completions (Zapier compatible)
- No dark mode
- No admin features on mobile
- Global availability, no regional restrictions

## Branded Mobile App
- Separate offering on select Plus plans
- Custom branding: colors, logo, app icon
- Published under your own App Store / Play Store listing
- In-App Purchases support
- More customization than standard Thinkific mobile app
- Your brand only — students don't see Thinkific branding

## Competitor Mobile Capabilities
- **LearnWorlds**: Native apps, branded, offline access supported
- **Kajabi**: Branded mobile app, no offline
- **Teachable**: No native app, responsive web only
- **Podia**: No native app, responsive web
- **Docebo**: Enterprise mobile with offline sync, microlearning, AI recommendations
- **Absorb**: Native app with offline, branded
- **TalentLMS**: Native app with offline
- **Moodle**: Open-source mobile app, offline support

## Key Stats
- 70%+ of online learning touches mobile devices
- Push notifications increase engagement significantly
- Offline access critical for field workers, traveling professionals
- App store presence builds credibility and discoverability
