# Thinkific Integrations Research Notes

## Key Facts
- Thinkific App Store: 96+ native integrations
- Zapier: Official app with triggers (enrollments, orders, completions) and actions (create enrollments, users)
- V2 REST API: users, courses, enrollments, orders, webhooks, products, groups, coupons
- V2 Webhooks: order, enrollment, user, lesson completion events
- SSO/SAML: Available on Plus plan
- No LTI support (enterprise feature — Docebo, Absorb, TalentLMS have it)
- No SCORM/xAPI on standard plans (Plus only for SCORM)

## Competitor Comparison
- Kajabi: 185+ native integrations, best Zapier depth, no LTI
- Docebo: 400+ connectors via Connect, deep Salesforce, LTI 1.3, SCIM
- Absorb: Enterprise-grade, LTI 1.3, Salesforce
- LearnWorlds: 50+ integrations, Zapier, no LTI
- Teachable: ~30 integrations, basic Zapier
- Podia: ~15 integrations, no public API

## Key Integration Categories
- **Marketing**: Mailchimp, ConvertKit, ActiveCampaign, HubSpot
- **CRM**: Salesforce (via Zapier for expert platforms, native for enterprise)
- **Payments**: Stripe, PayPal (TCommerce native)
- **Analytics**: Google Analytics, Facebook Pixel
- **Webinars**: Zoom (native on all plans)
- **Automation**: Zapier, Make
- **SSO**: SAML on Plus, social login on all plans
