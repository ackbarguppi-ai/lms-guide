# Thinkific Pricing Research Notes

## Current Plans (2026)
| Plan | Monthly | Annual | Key Features |
|------|---------|--------|-------------|
| Basic | $49/mo | $36/mo | Unlimited courses, 10K students, 1 community, 5 coaching sessions |
| Start | $99/mo | $74/mo | + Certificates, assignments, payment plans, memberships, unlimited coaching |
| Grow | $199/mo | $149/mo | + Remove branding, 3 communities, analytics, group orders, API |
| Plus | Custom | Custom | Unlimited students, SCORM, branded app, SSO/SAML |

## Transaction Fees
- TCommerce: 0% on all plans (standard Stripe ~2.9% + 30¢)
- Own Stripe surcharge: 5% Basic, 2% Start, 1% Grow
- No free plan anymore — 14-day trial only

## Hidden Costs
- 10,000 student cap on Basic/Start/Grow
- Stripe surcharge if not using TCommerce
- Branded Mobile app only on select Plus plans
- SCORM only on Plus
- SSO/SAML only on Plus
- Advanced API only on Grow+

## Competitor Pricing
- **Kajabi**: $69-$319/mo (annual), 0% tx fees, no free plan
- **Teachable**: Free plan (10% tx), $29/mo Basic (5%), $79/mo Pro (0%)
- **LearnWorlds**: $24/mo Starter ($5/sale), $79/mo Pro (0%), $249/mo Business
- **Podia**: Free (8% tx), $33/mo Mover (0%), $75/mo Shaker
- **Docebo**: Custom enterprise pricing, typically $10K+/year
- **Absorb**: Custom pricing, $800+/mo typical

## Key Differentiators
- Thinkific TCommerce: tax remittance included
- Annual billing saves 25-27%
- No coupon codes (affiliate sites are fake)
